# 🥗 푸드케어 렌즈 (FoodCare Lens) — MVP v2

기저질환자(당뇨·고혈압·신장질환·통풍)를 위한 **음성+자막 식단 안내 앱**.
성분표/음식 사진을 찍으면 **선택한 질환 전부에 대해 동시에** 신호등 판정을 내리고,
날짜별로 **누적 식단표**를 자동 작성합니다. PWA라서 폰 홈 화면에 앱처럼 설치됩니다.

## 실행 방법

- **가장 쉬움**: 프로젝트 상위 폴더의 `실행.bat` 더블클릭 (서버 시작 + 브라우저 자동 열림)
- 수동:

```bash
python -m http.server 8123 --directory foodcare-lens
# 브라우저에서 http://localhost:8123 접속
```

- **카메라·음성은 localhost 또는 https에서만 동작**합니다 (브라우저 보안 정책).
- 휴대폰 테스트: 같은 와이파이에서 `http://PC아이피:8123` 접속 후,
  크롬 `chrome://flags` → "Insecure origins treated as secure"에 주소 등록(개인 테스트용),
  또는 `npx localtunnel --port 8123` 등으로 https 터널.
- **앱으로 설치(PWA)**: 크롬 주소창 메뉴 → "홈 화면에 추가" (manifest + service worker 포함)

## v2 핵심 기능

| 기능 | 설명 |
|------|------|
| 다질환 동시 판정 | 홈에서 당뇨·고혈압·신장·통풍 칩을 여러 개 선택 → 한 끼를 질환별로 각각 신호등 판정. **선택 안 하면 일반 건강 모드**(당류+나트륨 일반 기준 600/1200mg, 하루 당류 50g) |
| 음성 + 자막 | 모든 안내는 TTS와 화면 하단 💬 자막 바에 **동시에** 표시 |
| 음식 사진 AI 분석 | Claude 비전(`claude-opus-4-8`, 구조화 출력)이 음식명·영양·퓨린 수준·섭취 조언 추정 |
| 성분표 AI 판독 | API 키가 있으면 성분표도 Claude 비전이 읽음 — 모든 영양소 추출 + 질환별 동시 판정 + 식단표 기록. 표에 없는 항목은 "판정 불가"로 정직하게 표시(0으로 오판 금지). 키 없거나 AI 실패 시 무료 OCR(Tesseract)로 자동 폴백 |
| 직관적 분량 | 열량을 밥공기로 환산 — 🍚=1공기(300kcal), 🥄=반 공기 |
| 식사 구분 | 기록 시 🌅아침/☀️점심/🌙저녁/🍪간식 선택 (시간대로 자동 추천) |
| 누적 식단표 | 날짜별 영구 저장, ◀▶ 날짜 이동, 최근 7일 주간 스트립(🟢🟡🔴 점) |
| 하루 식단 평가 | 질환별 하루 한도 대비 평가 + [당뇨]/[고혈압]/[신장]/[통풍] 사유 표시 |

## 판정 기준 (초기 가정값 — 학회/식약처 기준으로 보정 예정)

**한 끼(1회 제공량) 기준** — 중증도(경증 ×1.0 / 중등증 ×0.7 / 중증 ×0.5)로 하향:

| 질환 | 🟢 안심 | 🟡 주의 | 🔴 위험 |
|------|---------|---------|---------|
| 당뇨 (당류) | ≤5g | 5~15g | >15g |
| 고혈압 (나트륨) | ≤500mg | 500~1000mg | >1000mg |
| 신장 (나트륨) | ≤400mg | 400~800mg | >800mg |
| 통풍 (퓨린) | 낮음 | 보통 | 높음 |
| 일반 — 질환 없음 (나트륨) | ≤600mg | 600~1200mg | >1200mg |

**일반 모드**: 질환을 선택하지 않으면 당류(당뇨와 같은 5/15g)+나트륨(위 일반 기준)을 하나의
신호등으로 종합하고, 하루 한도는 당류 50g(WHO 일반)·나트륨 2000mg. **중증도는 적용하지 않습니다.**

**하루 누적 한도**: 당류 25g(WHO) · 나트륨 2000mg(WHO, 신장 1800mg) · 신장 단백질 60g ·
통풍 고퓨린 2회↑ 위험 · 열량 2000kcal. 70% 초과 시 🟡 주의.

## 안전망 (설계의 핵심 결정)

- OCR 실패 / 카메라 불가 / 오프라인 → 항상 **수동 입력** 경로로 안내
- AI 결과는 항상 "추정치이므로 참고만 하세요"를 음성·자막에 포함
- 첫 실행 1회 면책 고지: "의료 조언이 아닌 참고용, 최종 판단은 의사와"
- 색맹 대비: 색+이모지(🟢🟡🔴)+글자 라벨을 항상 병기

## ⚠️ API 키 주의

AI 분석은 홈 하단 **🔑 AI 설정**에서 입력한 API 키를 **이 기기 localStorage**에만 저장합니다.
**Gemini(무료) 또는 Claude 키 아무거나** 넣으면 키 접두사로 자동 인식합니다 —
`AIza...`는 Gemini, `sk-ant-...`는 Claude. Gemini 무료 키는 aistudio.google.com/apikey에서 발급(무료 등급).
키는 기기별 저장이므로 **폰에서 쓰려면 폰에서 한 번 입력**해야 합니다. **키가 든 상태로 페이지를 공유/배포하면 키가 노출됩니다** —
공개 배포 시에는 키를 서버 측으로 옮기는 프록시 구조가 필수입니다.
